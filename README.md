# Age of war

## Description

---


- 🔊프로젝트 소개

  Age of war는 동명의 유명 플래시게임의 모작입니다. 원작의 밸런스를 그대로 채용하였으며, 무엇보다 원작을 그대로 구현하는 것에 중점을 두었습니다.

       

- 개발 기간 : 2024.01.16 - 2024.01.23

- 🛠️사용 기술

   -언어 : C#

   -엔진 : Unity Engine

   -데이터베이스 : 로컬

   -개발 환경: Windows 10, Unity 2021.3.10f1
